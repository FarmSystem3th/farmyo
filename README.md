2️⃣ **리눅스 마스터**

<br>

3️⃣ **스터디 규칙**

## ✏️ 과제하는 방법

### 파일명 설정
- 각 장 별로 디렉토리가 이미 구성되어 있습니다. 
- 해당 장 안에서 자신의 마크다운파일을 생성해주세요. 
- `양식: 이름.md `
- (ex. `김팜요.md`)

### 커밋 규칙
- `[N장] 장 제목 - 이름`
- (ex. `[1장] 입력 값 조작 공격 - 김팜요`)
